# 분과별 요약 니즈 — 내과 + 세부 9분과 (research-im, 2026-07-04)

### 내과 (일반/Hospitalist)
- **차트에서 먼저 보는 것**: 활력징후 추이·야간 이벤트(sign-out) → 새 critical/pending 검사값 → I/O·체중 추이(수액 상태) → 활성 문제목록(중증도 순) → 최근 투약 변경(항응고제·인슐린·스테로이드) → code status/goals of care
- **요약 블록**: ① one-liner(나이·성별·입원사유) ② 문제목록(중증도순) ③ 지난 24시간 인터벌 변화 ④ 활력징후·I/O 추이 ⑤ 주요 검사·영상 추이 ⑥ 투약 변경 로그 ⑦ 미결 검사/컨설트 회신 ⑧ disposition & code status
- **Cannot-miss**: code status 미기재, critical/pending lab 누락, 신기능 기반 용량조절 누락, 컨설트 권고사항 미이행
- **포커스 칩**: "오늘 변경사항만" · "문제목록 우선순위" · "투약 변경 확인" · "다음 액션 아이템"
- **근거**: https://residencyadvisor.com/resources/med-school-life/efficient-pre-round-routine-a-30-minute-template-that-scales · https://www.ncbi.nlm.nih.gov/pmc/articles/PMC11430516/ · https://www.medrxiv.org/content/10.1101/2022.04.13.22273438v2.full

### 소화기내과
- **먼저 보는 것**: 간합성능(INR·알부민)·MELD/Child-Pugh 추이 → 최근 내시경 소견(정맥류·궤양·용종·조직검사) → Hb 추이·수혈력(출혈 이력) → 복수/뇌증 여부 → 항응고·항혈소판제(출혈위험) → 복부영상(종괴·담도·췌장)
- **요약 블록**: ① 진단명(간질환 원인/IBD 아형) ② 최근 내시경·영상 소견 ③ MELD/Child-Pugh 추이 ④ 출혈 이력·Hb 추이 ⑤ 복수/뇌증 등 간경변 합병증 ⑥ 영양·체중 변화 ⑦ 투약(항응고제·이뇨제·PPI) ⑧ 예정 procedure/추적계획
- **Cannot-miss**: 활동성 GI출혈 징후, 정맥류 출혈 이력, 간성뇌증 악화, 중증 급성췌장염 소견
- **포커스 칩**: "간기능 추이만" · "최근 내시경 소견" · "출혈 위험 평가" · "복수·뇌증 체크"
- **근거**: https://pmc.ncbi.nlm.nih.gov/articles/PMC8311468/ · https://www.ncbi.nlm.nih.gov/pmc/articles/PMC10278970/ · https://www.ncbi.nlm.nih.gov/pmc/articles/PMC11421874/

### 순환기내과
- **먼저 보는 것**: EF/echo 소견(추이) → troponin·BNP 추이 → EKG 리듬/허혈 변화 → 항혈전제 현황·위험점수(CHA2DS2-VASc·HAS-BLED) → 관상동맥/판막/디바이스 병력 → 심부전 stage/NYHA class
- **요약 블록**: ① 심장진단명·EF(HFrEF/HFpEF) ② troponin·BNP 추이 ③ EKG/리듬 이상 ④ 관상동맥·구조적 심질환 병력(PCI/CABG/판막) ⑤ 항혈전제 현황·위험점수 ⑥ 디바이스(pacemaker/ICD) ⑦ 최근 decompensation/입원 ⑧ 다음 검사·추적계획
- **Cannot-miss**: ACS 시사 소견, 새로운 부정맥, 항응고제 중단/미투약, decompensation 조기징후(체중증가·호흡곤란), 디바이스 이상
- **포커스 칩**: "EF·BNP 추이" · "항혈전제 현황" · "최근 decompensation" · "디바이스 상태"
- **근거**: https://www.soapnoteai.com/soap-note-guides-and-example/cardiology/ · https://thecardsnp.com/2019/09/05/tips-for-writing-a-cardiovascular-history/ · https://www.rhythm360.io/blog/signs-of-worsening-heart-failure

### 호흡기내과
- **먼저 보는 것**: 산소요구량 추이(재택산소 포함) → PFT(FEV1/FVC/DLCO) 추이 → 흉부영상(CT) 변화 → 흡연력(pack-years) → 최근 악화/입원 빈도 → 감염/배양 결과
- **요약 블록**: ① 진단명·중증도(COPD/천식/ILD/폐암) ② PFT 추이 ③ 영상 소견 추이(결절·섬유화·폐렴) ④ 산소요구량/재택산소 ⑤ 흡연력·노출력 ⑥ 악화 이력 빈도 ⑦ 투약(흡입기·스테로이드·생물학적제제) ⑧ 폐기능 예후/이식후보 여부
- **Cannot-miss**: 급성 저산소증, 새로운 폐결절/종괴, 스테로이드 장기사용 부작용, 반복 급성악화 패턴, 폐색전증 위험
- **포커스 칩**: "산소요구량 추이" · "PFT 변화" · "영상 신규소견" · "악화 이력"
- **근거**: https://document.resmed.com/documents/articles/1019404_Oxygen_Qualifying_Guidelines_amer_eng.pdf · https://www.medrxiv.org/content/10.1101/2020.10.30.20223511.full.pdf · https://pmc.ncbi.nlm.nih.gov/articles/PMC11332629/
