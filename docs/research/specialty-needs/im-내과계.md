# 분과별 요약 니즈 — 내과 + 세부 9분과 (research-im, 2026-07-04)

> 내과(Hospitalist) + 소화기·순환기·호흡기·내분비대사·신장·혈액종양·감염·알레르기·류마티스.

### 내과 (일반/Hospitalist)
- **먼저 보는 것**: 활력징후 추이·야간 이벤트(sign-out) → 새 critical/pending 검사값 → I/O·체중 추이 → 활성 문제목록(중증도순) → 최근 투약 변경(항응고제·인슐린·스테로이드) → code status/goals of care
- **요약 블록**: ① one-liner(나이·성별·입원사유) ② 문제목록(중증도순) ③ 지난 24h 인터벌 변화 ④ 활력징후·I/O 추이 ⑤ 주요 검사·영상 추이 ⑥ 투약 변경 로그 ⑦ 미결 검사/컨설트 회신 ⑧ disposition & code status
- **Cannot-miss**: code status 미기재, critical/pending lab 누락, 신기능 기반 용량조절 누락, 컨설트 권고 미이행
- **포커스 칩**: "오늘 변경사항만" · "문제목록 우선순위" · "투약 변경 확인" · "다음 액션 아이템"
- **근거**: residencyadvisor.com/…/efficient-pre-round-routine · ncbi.nlm.nih.gov/pmc/articles/PMC11430516/ · medrxiv 2022.04.13.22273438v2

### 소화기내과
- **먼저 보는 것**: 간합성능(INR·알부민)·MELD/Child-Pugh 추이 → 최근 내시경 소견(정맥류·궤양·용종·조직검사) → Hb 추이·수혈력(출혈) → 복수/뇌증 → 항응고·항혈소판제 → 복부영상(종괴·담도·췌장)
- **요약 블록**: ① 진단명(간질환 원인/IBD 아형) ② 최근 내시경·영상 ③ MELD/Child-Pugh 추이 ④ 출혈 이력·Hb 추이 ⑤ 간경변 합병증(복수/뇌증) ⑥ 영양·체중 ⑦ 투약(항응고제·이뇨제·PPI) ⑧ 예정 procedure
- **Cannot-miss**: 활동성 GI출혈, 정맥류 출혈 이력, 간성뇌증 악화, 중증 급성췌장염
- **포커스 칩**: "간기능 추이만" · "최근 내시경 소견" · "출혈 위험 평가" · "복수·뇌증 체크"
- **근거**: pmc PMC8311468 · pmc PMC10278970 · pmc PMC11421874

### 순환기내과
- **먼저 보는 것**: EF/echo 추이 → troponin·BNP 추이 → EKG 리듬/허혈 변화 → 항혈전제 현황·위험점수(CHA2DS2-VASc·HAS-BLED) → 관상동맥/판막/디바이스 병력 → 심부전 stage/NYHA
- **요약 블록**: ① 심장진단명·EF(HFrEF/HFpEF) ② troponin·BNP 추이 ③ EKG/리듬 이상 ④ 관상동맥·구조적 심질환(PCI/CABG/판막) ⑤ 항혈전제·위험점수 ⑥ 디바이스(pacemaker/ICD) ⑦ 최근 decompensation 이력 ⑧ 다음 검사
- **Cannot-miss**: ACS 시사 소견, 새 부정맥, 항응고제 중단/미투약, decompensation 조기징후(체중증가·호흡곤란), 디바이스 이상
- **포커스 칩**: "EF·BNP 추이" · "항혈전제 현황" · "최근 decompensation" · "디바이스 상태"
- **근거**: soapnoteai.com/…/cardiology · thecardsnp.com cardiovascular history · rhythm360.io worsening heart failure

### 호흡기내과
- **먼저 보는 것**: 산소요구량 추이(재택산소) → PFT(FEV1/FVC/DLCO) 추이 → 흉부CT 변화 → 흡연력(pack-years) → 최근 악화/입원 빈도 → 감염/배양
- **요약 블록**: ① 진단·중증도(COPD/천식/ILD/폐암) ② PFT 추이 ③ 영상 추이(결절·섬유화·폐렴) ④ 산소요구량/재택산소 ⑤ 흡연력·노출력 ⑥ 악화 빈도 ⑦ 투약(흡입기·스테로이드·생물학적제제) ⑧ 예후/이식후보
- **Cannot-miss**: 급성 저산소증, 새 폐결절/종괴, 스테로이드 장기 부작용, 반복 급성악화, 폐색전 위험
- **포커스 칩**: "산소요구량 추이" · "PFT 변화" · "영상 신규소견" · "악화 이력"
- **근거**: resmed oxygen qualifying guidelines · medrxiv pack-year 2020.10.30 · pmc PMC11332629

### 내분비대사내과
- **먼저 보는 것**: HbA1c·혈당 로그 추이 → 갑상선기능(TSH/FT4) 추이 → 인슐린/경구약 변경 → 스테로이드 사용력(부신저하) → 급성 대사이상 이력(DKA/HHS·thyroid storm·adrenal crisis)
- **요약 블록**: ① 당뇨 진단·합병증(신증/망막증/신경병증) ② HbA1c·혈당 추이 ③ 약물 변경 이력 ④ 갑상선기능 추이 ⑤ 부신/뇌하수체 축(해당시) ⑥ 지질·골대사 ⑦ 최근 급성대사 이벤트 ⑧ 다음 검사
- **Cannot-miss**: adrenal crisis, thyroid storm, DKA/HHS 재발, 스테로이드 급성 중단 부신부전, 반복 저혈당
- **포커스 칩**: "혈당·HbA1c 추이" · "갑상선기능 추이" · "약물 변경 이력" · "급성대사 이벤트"
- **근거**: pmc PMC8971272 · StatPearls NBK499968 (Adrenal Crisis) · medscape 567307

### 신장내과
- **먼저 보는 것**: Cr/GFR 추이(baseline 대비) → 전해질(특히 K+) → AKI vs CKD 감별(baseline Cr 필수) → 투석 여부/access → 단백뇨·혈뇨 → 신독성 약물/조영제 노출
- **요약 블록**: ① 신기능 진단(AKI/CKD stage)·원인 ② Cr/GFR 추이(baseline 대비) ③ 전해질 추이(K·HCO3) ④ 투석 방식·access·스케줄 ⑤ 단백뇨/혈뇨/영상 ⑥ 신독성·용량조절 약물 ⑦ 혈압·체액상태 ⑧ 다음 계획(투석 개시·신생검)
- **Cannot-miss**: 응급 고칼륨혈증, 급성투석 필요(요독증·폐부종·산증), AKI를 CKD baseline으로 오인, 조영제/신독성약물 노출 미확인
- **포커스 칩**: "Cr·GFR 추이" · "전해질(K+) 체크" · "투석 상태" · "신독성 약물 확인"
- **근거**: pmc PMC7448835 (KDIGO 고칼륨) · pmc PMC8573871 (baseline Cr) · pointofcaremedicine hyperkalemia

### 혈액종양내과
- **먼저 보는 것**: 병기·조직형 → 항암 레지멘·사이클·최근 영상 반응평가 → CBC 추이(ANC·Hb·혈소판) → ECOG → 항암 독성 이력 → 수혈력/성장인자
- **요약 블록**: ① 진단·병기·조직형 ② 치료이력(레지멘·사이클·최근 투여일) ③ 최근 영상 반응평가 ④ CBC 추이·ANC/혈소판 nadir ⑤ ECOG 변화 ⑥ 부작용/독성(등급) ⑦ 수혈력·성장인자 ⑧ 다음 계획
- **Cannot-miss**: 발열성 호중구감소증(ANC<1000+발열), 종양응급(척수압박·SVC·종양용해), 병기 진행/재발, 중증 항암독성
- **포커스 칩**: "치료이력·사이클" · "CBC·ANC 추이" · "영상 반응평가" · "독성·부작용 확인"
- **근거**: orbdoc.com oncology consult · pmc PMC12608793 (ECOG)

### 감염내과
- **먼저 보는 것**: 배양·감수성(혈액/소변/객담) → 현재/과거 항생제 이력·반응 → 면역저하(이식·항암·스테로이드·HIV) → 감염원/source control → 염증수치(CRP/procalcitonin) 추이 → 여행력/내성균 이력
- **요약 블록**: ① 감염진단/의심 병원체 ② 배양·감수성 추이 ③ 항생제 이력(투여일·변경사유·반응) ④ 면역상태 ⑤ 염증수치 추이 ⑥ Source control 상태 ⑦ 여행력·내성균 이력 ⑧ de-escalation/치료기간
- **Cannot-miss**: 패혈성 쇼크, 미해결 감염원, 내성균(MRSA/VRE/CRE) 이력 누락, 면역저하자 비전형 감염, pending 배양 누락
- **포커스 칩**: "배양·감수성 결과" · "항생제 이력" · "면역저하 상태" · "Source control 확인"
- **근거**: orbdoc.com ID consult · med.unc.edu ID curriculum · unmc sepsis de-escalation

### 알레르기내과
- **먼저 보는 것**: 약물/음식 알레르기 이력(반응유형·중증도) → 원인물질 검사(피부단자/특이 IgE) → 과거 중증반응·입원 이력 → 면역요법 이력·반응 → 동반 알레르기질환(천식·아토피·비염) 조절
- **요약 블록**: ① 진단·원인물질 ② 반응 이력(증상·중증도·타이밍) ③ 검사결과(피부시험/IgE) 추이 ④ 중증반응 병력·에피펜 처방 ⑤ 면역요법 진행 ⑥ 동반질환 조절 ⑦ 회피·대체 약물 ⑧ 응급대응계획(action plan)
- **Cannot-miss**: 중증 전신반응 병력 누락, 교차반응 약물/음식 미확인, 에피펜 처방·교육 여부, 베타차단제 복용 중 면역요법 위험
- **포커스 칩**: "원인물질·반응이력" · "중증반응 병력" · "면역요법 진행상황" · "응급대응계획"
- **근거**: worldallergyorganizationjournal WAO 2020 · annallergy 약물알레르기 · pmc PMC11996274

### 류마티스내과
- **먼저 보는 것**: 자가항체(RF·anti-CCP·ANA·anti-dsDNA) → 관절진찰(TJC/SJC)·질병활성도(DAS28) → 염증수치(ESR/CRP) 추이 → 면역억제제/생물학적제제 이력·반응 → Flare vs 감염 감별 → 장기침범(폐/신장/심장)
- **요약 블록**: ① 진단·아형 ② 자가항체 프로파일 ③ 질병활성도 추이(DAS28) ④ 염증수치 추이 ⑤ 치료이력(DMARD/생물학적·반응) ⑥ 장기침범 평가 ⑦ 감염 위험(면역억제 정도) ⑧ 다음 계획
- **Cannot-miss**: Flare와 감염 감별 실패, 생물학적제제 시작 전 잠복결핵/B형간염 스크리닝 누락, 장기침범(ILD·루푸스신염) 진행, 스테로이드 급감 flare
- **포커스 칩**: "질병활성도 추이" · "자가항체·염증수치" · "Flare vs 감염" · "장기침범 체크"
- **근거**: pmc PMC4067329 (DAS28) · creakyjoints 질병활성도 · pmc PMC6153569 (flare vs 감염)

## 공통 패턴 (10분과 관통)
1. 절대수치보다 **추이(trend)** 우선 — EF·BNP·Cr/GFR·HbA1c·CBC·DAS28 모두 "방향과 속도"가 의사결정.
2. Cannot-miss = 대부분 **급성 악화 또는 감별 실패**(감염 vs flare, AKI vs CKD 등) → 요약 최상단 고정.
3. 전 분과 **최근 투약 변경/중단**(항응고제·스테로이드·면역억제제·항생제) 필수 → "투약 변경 로그" 공통 컴포넌트.
